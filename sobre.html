Sobre o nosso projeto

O projeto consiste no desenvolvimento de um site que tem como objetivo transmitir informações sobre o crescimento de casos ocasionados pela COVID-19.

Nosso site será atualizado diariamente conforme os números foram subindo, e nossa principal intenção no meio de tudo isso, é mostrar que ainda devemos ter cuidado com o vírus.

Sentimos muito sobre todas as perdas e acreditamos que juntos, podemos superar isso :)

Ferramentas e Tecnologias

O projeto consistiu no uso do aplicativo Visual Studio Code, com as linguagens de programação HTML, CSS e Java Script.

Cada linha foi escrita utilizando os conhecimentos praticados em sala de aula.

Utilizamos ferramentas de paletas de cor, o google charts e o Canva para os desings.


Créditos
